# UniSocial
A social platform built to help university students connect via mutual interests/hobbies and features an integrated chat.

## Tech/framework used
* ReactJS
* NodeJS
* ExpressJS
* MongoDB
* JWT
* Amazon S3
* Socket.io

## Installation
#### Client
```
cd Client
npm install
npm start
```

#### Server
```
cd Server
npm install
node server.js
```
